# 🚀 一键构建 APK 完全指南

**不需要安装任何软件！不需要配置任何环境！**

---

## 📋 你需要准备什么

只需要：
1. ✅ 一个 GitHub 账号（免费）
2. ✅ 浏览器

**不需要：**
- ❌ 安装 Android Studio
- ❌ 安装 Java JDK
- ❌ 配置任何环境变量
- ❌ 任何编程知识

---

## 🎯 详细步骤（跟着做就行）

### 第1步：创建 GitHub 账号

1. 打开 https://github.com
2. 点击右上角 "Sign up"
3. 按提示填写邮箱、密码、用户名
4. 完成注册

**如果你已有 GitHub 账号，跳过这步**

---

### 第2步：创建新仓库

1. 登录 GitHub 后，点击右上角 "+" 号
2. 选择 "New repository"
3. 填写：
   - Repository name: `step-analyzer-android`
   - Description: `STEP模具分析 Android应用`
   - 选择 "Public"
4. 点击 "Create repository"

---

### 第3步：上传项目文件

1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 将以下文件夹拖拽上传：
   ```
   /vol1/1000/Qwen/step-analyzer-android/
   ```
3. 等待上传完成
4. 点击 "Commit changes"

**上传内容包括：**
- `app/` 文件夹（源代码）
- `build.gradle`（构建配置）
- `.github/` 文件夹（自动构建配置）
- 其他配置文件

---

### 第4步：自动构建开始！

1. 上传完成后，自动构建立即开始
2. 点击仓库页面的 "Actions" 标签
3. 你会看到构建正在运行（黄色圆圈）
4. 等待 3-5 分钟，构建完成（绿色对勾 ✅）

---

### 第5步：下载 APK

1. 在 "Actions" 页面，点击已完成的构建
2. 向下滚动，找到 "Artifacts" 部分
3. 点击 `step-analyzer-apk` 下载
4. 得到一个 `step-analyzer-apk.zip` 文件
5. 解压后得到 `app-debug.apk`

**APK 文件位置：**
```
step-analyzer-apk.zip
└── app-debug.apk  ← 就是这个！
```

---

### 第6步：安装到手机

**方法1：直接安装**
1. 将 `app-debug.apk` 传输到手机
2. 在手机上打开文件管理器
3. 点击 APK 文件
4. 按提示安装

**方法2：使用微信/QQ传输**
1. 电脑登录微信/QQ
2. 发送 APK 文件给自己
3. 手机上接收并安装

**方法3：使用网盘**
1. 上传到百度网盘/阿里云盘
2. 手机下载并安装

---

## ⚠️ 安装时的注意事项

### 手机提示"未知来源"
1. 点击 "设置"
2. 找到 "安全" 或 "隐私"
3. 开启 "允许安装未知来源应用"

### 安装后无法打开
1. 检查 Android 版本是否 ≥ 7.0
2. 重新下载 APK 再安装

## ❓ 常见问题

### Q: 构建失败了怎么办？
**A:** 点击 Actions 标签，查看错误日志。通常是因为文件没上传完整，检查是否所有文件都上传了。

### Q: 下载的 APK 无法安装？
**A:** 
1. 确保手机开启了"未知来源"权限
2. 检查 Android 版本是否 ≥ 7.0
3. 重新下载 APK

### Q: 可以给别人安装吗？
**A:** 可以！APK 文件可以分享给任何人，直接安装即可。

### Q: 需要付费吗？
**A:** 完全免费！GitHub Actions 提供每月 2000 分钟免费构建时间。
